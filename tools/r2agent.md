# r2agent

## Help

      Usage: r2agent [-adhs] [-p port]
       -a listen for everyone (localhost by default)
       -d run in daemon mode (background)
       -h show this help message
       -s run in sandbox mode
       -p [port] specify listening port (defaults to 8080)
